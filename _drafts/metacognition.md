Structure idea: walk through each compression stage

The compression mechanism:
- Courage: willingness to kill directions. Deliberate. Slow. Requires overriding attachment.
  - What makes this hard: sunk cost, ego, narrative momentum
  - What makes this possible: the loop provides evidence fast enough to overcome attachment
- Judgment: pattern recognition from repeated courageous filtering.
  - Courage applied enough times → the pattern becomes recognizable before the evidence arrives
  - "I've killed this type of direction before, I can see it earlier now"
  - This is the filter sharpening itself through use
- Intuition: judgment compressed below conscious access.
  - Judgment applied enough times → the decision happens before deliberation
  - The slot operates automatically. No longer requires attention.
  - This frees attention for the next layer of problems

Kaizen/hansei mapping:
- Sort: identify what belongs and what doesn't (filter)
- Sweep: clean out accumulated noise (consolidation)
- Standardize: make the improved process the default (remember)
- Each applied to cognition, not manufacturing

The math connection:
- Each compression step increases the rate constant in the sigmoid
- Courage → judgment: the filter gets faster, so cycles/time increases
- Judgment → intuition: the filter becomes automatic, so attention freed for higher-order filtering
- This is why the rate depends on accumulated capability (the sigmoid derivation)
